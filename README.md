# Converso-Video-Audio-em-Python
Um conversor leve e eficiente em Python para transformar URLs do YouTube em arquivos de áudio (MP3) e vídeo (MP4).
# 🎥 TubeTailor: YouTube to Media Converter

O **TubeTailor** é uma ferramenta simples e eficiente escrita em Python que permite converter links do YouTube diretamente para arquivos de vídeo (**MP4**) ou apenas áudio (**MP3**).

Ideal para quem busca rapidez e quer evitar sites cheios de anúncios e malwares.

---

## ✨ Funcionalidades

* **Download de Vídeo:** Baixa vídeos em alta resolução no formato MP4.
* **Extração de Áudio:** Converte o vídeo automaticamente para MP3 com excelente qualidade.
* **Simplicidade:** Interface via terminal ou script direto (paste & run).
* **Leveza:** Consome poucos recursos do sistema.

## 🚀 Tecnologias Utilizadas

* [Python 3.x](https://www.python.org/)
* [yt-dlp](https://github.com/yt-dlp/yt-dlp) (ou `pytube`, ajuste conforme sua biblioteca)
* [FFmpeg](https://ffmpeg.org/) (necessário para conversão de áudio)
